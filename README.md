# Todo
[![Build Status](https://travis-ci.org/opnay/Todo.svg?branch=master)](https://travis-ci.org/opnay/Todo)

Todo list application for android with kotlin.

- Min SDK: 21(5.0 Lollipop)
- Target SDK: 27(8.1 Oreo)
